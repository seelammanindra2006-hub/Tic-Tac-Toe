# 🎮 Tic Tac Toe

A modern and responsive **Tic Tac Toe** web application built using **HTML, CSS, and JavaScript**. This project provides an interactive two-player gaming experience with a clean user interface and smooth gameplay.

---

## 📌 Features

* 🎯 Two-player gameplay (Player X & Player O)
* 🏆 Automatic winner detection
* 🤝 Draw detection
* 📊 Live scoreboard
* ✨ Winning cell highlight
* 🔄 Restart game option
* 🗑️ Reset score functionality
* 📱 Fully responsive design
* 🎨 Modern glassmorphism user interface

---

## 🛠️ Technologies Used

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any modern web browser.
4. Start playing Tic Tac Toe.

---

## 🎮 How to Play

1. Player **X** starts the game.
2. Players take turns placing **X** and **O** on the 3×3 board.
3. The first player to align three symbols horizontally, vertically, or diagonally wins.
4. If all cells are filled without a winner, the game ends in a draw.
5. Use the **Restart Game** button to start a new round.
6. Use the **Reset Score** button to clear all scores.
